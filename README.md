## Auto-Invoice Analysis | UiPath Agentic Automation

An agentic RPA workflow built in UiPath Studio Cloud.

### What it does:
- Monitors for new invoice uploads
- Downloads and extracts invoice data  
- AI agent analyses and makes approval decision
- ✅ Approved invoices → sent to Approved Google Drive folder
- ❌ Rejected invoices → sent for human review
- Post-review analysis for declined invoices

### Tools Used:
UiPath Studio Cloud, UiPath Orchestrator, Google Drive
